## E-commerce Sales Dashboard – $1.46M Full Analysis

Interactive **Power BI** dashboard analyzing **$1.46 million** in sales across **5,654 orders** from a multi-category e-commerce business.

### Key Business Insights
- **Total Sales**: **$1,456,309** from **5,654 orders**
- **Average Order Value (AOV)**: **$1,456**
- **Top Category**: **Sports** → **$318K** (22%), followed by Electronics ($294K) and Clothing ($288K)
- **Highest spending segment**: Customers **46+ years old**
- **Top 10 customers** contribute **over $47K** in revenue
- **Peak sales day**: **Friday**
- **Preferred payment method**: **Credit Card** → **$398K** (27% of total)
